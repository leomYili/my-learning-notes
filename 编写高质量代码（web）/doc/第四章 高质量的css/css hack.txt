
1.ie条件注释法
  因为ie在css的解析上和其他浏览器脑子不一样。绝大多数csshack都是针对ie进行的。而微软则推出官方hack方法，——ie条件注释法
  <!--[if IE]>
    <link>
  <![endif]-->
  如果想针对某个特定版本的ie浏览器，if IE 6、7、8
  如果我们想针对某个范围以内版本的ie进行hack，可以结合lte（小于等于）\lt（小于）\gte（大于等于）\gt（大于）\!（不等于）关键字来进行注释
2.选择符前缀法
  是另一种常用的hack方法，原理是在css选择符前加一些只有特定浏览器才能识别的前缀，例"*html"针对ie6；
  "*+html"针对ie7，这种方法在向后兼容性上存在一定风险，另外，选择符前缀法不能用于内联样式上，因为无法使用选择符；
3.样式属性前缀发
  该方法是在样式之前添加前缀,例"_"只在ie6生效,"*"在ie6和ie7生效。*width:70px  _width:70px
